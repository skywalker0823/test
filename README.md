# task3 資料庫使用方法
0. 虛擬環境
1. 安裝套件 `pip install -r requirements.txt`
2. 啟動mysql 容器 : docker run -dp 3306:3306 --name a_mysql_local -e MYSQL_ROOT_PASSWORD=123123 mysql:8
3. 建立資料庫 : python3 test3/build.py
4. 執行 : python3 test3/data.py
